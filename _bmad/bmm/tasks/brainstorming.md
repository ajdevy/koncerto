# Brainstorming Task

**Purpose:** Generate ideas and explore possibilities before defining requirements

## When to Use
- Starting a new project
- Exploring new features
- Solving complex problems
- Creative ideation sessions

## Process

1. **Set Context**
   - Share project background
   - Define the problem space
   - Identify constraints

2. **Generate Ideas**
   - Use open-ended questions
   - Encourage wild ideas
   - Build on others' suggestions
   - No criticism allowed

3. **Organize Ideas**
   - Group related ideas
   - Identify themes
   - Remove duplicates
   - Prioritize by impact

4. **Document Results**
   - Create Idea Summary
   - Define next steps
   - Assign owners

## Output Format

```markdown
# Brainstorming Session: [Topic]

## Participants
- [List participants]

## Problem Statement
[Clear problem definition]

## Ideas Generated
### Theme 1: [Theme Name]
- Idea 1.1
- Idea 1.2

### Theme 2: [Theme Name]
- Idea 2.1
- Idea 2.2

## Top Priorities
1. [Highest priority idea]
2. [Second priority]
3. [Third priority]

## Next Steps
- [ ] [Action item 1]
- [ ] [Action item 2]
```
